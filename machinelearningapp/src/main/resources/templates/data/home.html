Easy Machine Learning

Create a linear regression model of a dataset through machine learning.

  1. Enter your own dataset (or use a sample dataset)

    2. Because a linear model will be trained, Y data should be continuous, rather than discrete. Our site supports only univariate linear regression, so X data should be single-factor.

  2. View a graph of the dataset

    3. This will give you a sense of the shape of your data.

  3. Set the parameters for machine learning

    4. Or use the default parameters for the gradient descent algorithm.

  4. Receive a trained model

    5. You will see its metrics and a graph of its predictions.

  5. Repeat steps 3 and 4 as desired

    6. Play with the parameters until you've trained a good model.

  6. Test your model

    7. See how your model performs on the testing set.

  7. Make predictions

    8. Use your model to make predictions about incoming data.
